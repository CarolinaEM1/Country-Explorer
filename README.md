# Country Explorer 🌎

Country Explorer is a web application that allows users to explore country information using data stored in Supabase.

## Features

- OAuth authentication with GitHub
- Data persistence using Supabase PostgreSQL
- Data loaded from REST Countries API
- Responsive card-based interface
- Country search and filtering (in progress)

## Technologies

- React
- Vite
- Supabase
- REST Countries API
- JavaScript
